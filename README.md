Work in progress knowledge quest to convert Might & Magic 3 / III Isles of Terra game data to the data format used in Might & Magic 4/5 Xeen engine.

High level workflow proposal to apply the conversion:
- Extract MM3 data files
- Extract MM4 data files
- Run conversion script(s) to convert the MM3 data and overlay onto the MM4 files
- Repack the updated MM4 files folder to XEEN.CC (using Xeen CC Packer)
- Run the total conversion "mod" in Xeen engine and/or ScummVM
