Map structure conversion:
MM4/5:
MM4 (XEEN): MAZE0001~MAZE0085 ~ MAZEX100~MAZEX142
MM5 (DARK): MAZE0001~MAZE0099 ~ MAZEX100~MAZEX183
SWORDS:		MAZE0001~MAZE0099
- DARK0001.TXT	text file with name of map only "Map A1"+0x00(null terminated)
- XEEN0001.TXT	"" for clouds levels

- MAZE0001.EVT	event file ~3kb
- MAZE0001.DAT	892 bytes
- MAZE0001.MOB	416 bytes
- AAZE0001.HED	padding file? just 020100 repeating, 2048 bytes
- AAZE0001.TXT	strings file

MM3:
MAZE01~MAZE106
- MAZE01.DAT	832 bytes; from SAV file!
- MAZE01.EVT	event file ~2kb; from SAV file!
- MAZE01.MOB	138 bytes; from SAV file!
- text01.maz	strings file; from CC file


- _UNKNOWN_FILE_XXXX.BIN	strings files / map data?

MM3 sprite format differences:
- CMD 0: same as MM4 CMD 0 (len + 1 bytes are palette indexes)
- CMD 1: also same as MM4 CMD 0 (len + 1 bytes are palette indexes)
- CMD 2: NOT SURE... STOP?
- CMD 3: NOT SURE (stream copy?)
- CMD 4: same as MM4 CMD 5 (skip len + 1 pixels)
- CMD 5: same as MM4 CMD 5 but skip len + 33 pixels (like MM4 CMD 1)
- CMD 6: same as MM4 CMD 2 (draw len + 3 times)
- CMD 7: NOT SURE; same as MM4 pattern?
